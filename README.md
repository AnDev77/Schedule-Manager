# Schedule-Manager

## 설치 방법

### 전체 프로젝트 설치

```console
npm install
```

혹은

```console
npm run install:all
```

### 백엔드만 설치

```console
npm run install:backend
```

### 프론트엔드만 설치

```console
npm run instal:frontend
```

## 실행 방법

### 프론트엔드 + 백엔드 실행

```console
npm run dev
```

### 프론트엔드만 실행

```console
npm run frontend
```

### 백엔드만 실행

```console
npm run backend
```
